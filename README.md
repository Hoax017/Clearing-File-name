# File-name-Cleaner

THIS TOOL USE PHP

## launch
```bash
./clearfilename <directory>
```
EXEMPLE :
```bash
./clearfilename G:\Film
```
If no directory selected it select directory form config.json
```JSON
{
    "default_folder": "\/home\/Hoax017\/Download"
}
```
